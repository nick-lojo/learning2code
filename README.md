# learning2code

Self-directed software engineering curriculum — 72-week structured program.

## About

Building the necessary skills to be a proficient engineer in the AI-era.
This repository documents every lesson, practice problem, and project built along the way.

**Current Phase:** Phase 1 — Python Fundamentals (Python Crash Course, 3rd Ed.)

---

## Structure

Folders follow this naming convention:

```
week##_BOOK_chapter##/                → Lesson files + TIY exercises
week##_BOOK_chapter##/practice/       → Practice problems
week##_BOOK_chapter##_mini_project/   → Chapter mini-project
```

Each chapter has three components:
- **Lesson** — concept notes + coded examples, section by section
- **Practice** — independent problems, one concept at a time
- **Mini-Project** — small working system built from scratch
